Getting Started with vbuf

Minimal setup guide for creating a vbuf editor instance.

1. Include the Script

<script src="vbuf.js"></script>

2. Required HTML Structure

Important: All these elements are required. Missing any will cause runtime errors. 

<blockquote class="wb no-select" tabindex="0" id="editor">
    <textarea class="wb-clipboard-bridge" aria-hidden="true"></textarea>
    <div style="display: flex">
        <div class="wb-gutter"></div>
        <div class="wb-lines" style="flex: 1; overflow: hidden;"></div>
    </div>
    <div class="wb-status" style="display: flex; justify-content: space-between;">
        <div class="wb-status-left"><span class="wb-linecount"></span></div>
        <div class="wb-status-right">
            <span class="wb-coordinate"></span>
            <span>|</span>
            <span class="wb-indentation"></span>
        </div>
    </div>
</blockquote>

Required Elements Checklist:

3. Initialize the Editor

const element = document.getElementById('editor');
const editor = new Vbuf(element, {
    initialViewportSize: 15,
    colorPrimary: "#B2B2B2",
    colorSecondary: "#212026"
});

// Set initial content
editor.Model.text = "Hello, World!";

// Focus the editor
element.focus();

4. Live Example
