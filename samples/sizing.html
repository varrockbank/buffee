Editor Sizing with Character Units

The editor uses monospace font. Set width on the container element (.wb) via CSS. Use ch units for predictable column counts, or 100% to fill parent.

Height (number of lines) is set via the initialViewportSize option in the constructor.

80 columns × 10 lines

CSS: #editor-80 { width: 80ch; }

|

1 line editor (no status)

JS: initialViewportSize: 1, showStatusLine: false

|

40 columns (no gutter)

CSS: #editor-40-clean { width: 40ch; } + showGutter: false

|

40 columns × 5 lines

CSS: #editor-40 { width: 40ch; }

|

120 columns × 15 lines

CSS: #editor-120 { width: 120ch; }

|

Thick border (3ch) × 6 lines

CSS: #editor-thick { width: 60ch; border: 3ch solid #ff6b35; }

|

100% width × 8 lines

CSS: #editor-full { width: 100%; }
