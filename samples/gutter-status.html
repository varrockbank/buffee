Gutter (Line Numbers) and Status Bar

Control visibility via constructor options: showGutter and showStatusLine

Default (gutter + status bar)

JS: new Vbuf(el, { showGutter: true, showStatusLine: true })

|

No gutter (line numbers hidden)

JS: new Vbuf(el, { showGutter: false })

|

No status bar

JS: new Vbuf(el, { showStatusLine: false })

|

Minimal (no gutter, no status bar)

JS: new Vbuf(el, { showGutter: false, showStatusLine: false })

|
